# 🌱 Green Coding Score - VS Code Extension

A VS Code extension to measure the sustainability of AI prompts.

## 🚀 Features

- Analyze prompt efficiency
- Estimate token usage
- Calculate Green Coding Score
- Visual feedback inside VS Code

## 🧪 Demo

Run command:
> Analyze Prompt Sustainability

## 🔗 Backend

Requires Green Coding Score API running locally.

## 🌍 Vision

Bring sustainability awareness into developer workflows.
